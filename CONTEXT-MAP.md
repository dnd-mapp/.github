# Context Map

This repo (`dnd-mapp/.github`) is the org's central home for cross-repo, org-wide documentation. It currently holds two unrelated contexts.

## Contexts

- [Issue Tracking](./docs/issue-tracking/CONTEXT.md): the org's Issue Type system (Epic, Story, Task, Bug, Spike) and how it's tracked across repos
- [Release Tooling](./docs/release-tooling/CONTEXT.md): shared vocabulary for how `dnd-mapp` repos tag, validate, and publish releases

## Relationships

- No shared vocabulary between the two contexts. They're unrelated domains that happen to share this org-wide `.github` repo as their central home, rather than living in any single product repo.

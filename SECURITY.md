# Security Policy

## Supported versions

Only the latest release of each dnd-mapp project is supported for security fixes.

## Reporting a vulnerability

Use GitHub's **Report a vulnerability** button on the Security tab of the affected repository. This opens a private advisory — do not open a public issue for security concerns.

I'll aim to acknowledge reports promptly, but no response time is guaranteed.

## Disclosure

Once a fix is available, a GitHub Security Advisory will be published publicly. Reporters are credited in the advisory unless they prefer to remain anonymous.

# Contributing

dnd-mapp is a platform for Dungeon Masters and Players to manage their Dungeons & Dragons 5th Edition campaigns.

## This is a closed project

Forking is disabled and pull requests are restricted to maintainers. Unsolicited pull requests will not be accepted.

## How to participate

### Report a bug

Open an issue using the **Bug** template. Include steps to reproduce, expected behavior, actual behavior, and your environment. See the template for the full list of required fields.

### Ask a question

Post in [Discussions → Q&A](https://github.com/orgs/dnd-mapp/discussions/categories/q-a) for questions about the platform or the codebase.

### Suggest an idea

Post in [Discussions → Ideas](https://github.com/orgs/dnd-mapp/discussions/categories/ideas) to propose a feature or improvement. Suggestions are reviewed at the maintainer's discretion — no response time is guaranteed. Ideas that are taken up may be converted into Epics, Stories, or Tasks.

## Security

To report a security vulnerability, see [SECURITY.md](SECURITY.md). Do not open a public issue for security concerns.
